# FAA RF Interference Analysis Tool
### For ITU-R WP Policy Support

A browser-based RF engineering tool to support FAA spectrum protection 
work in ITU-R proceedings. Built to help RF engineers supporting NTIA/FAA 
policy assess interference threats to protected aeronautical frequencies.

## Modules
- 📡 **Protected Bands** — FAA frequency database with protection criteria
- 🔗 **Link Budget** — Full Friis link budget with waterfall chart
- 📊 **Noise & I/N** — Noise floor and interference-to-noise analysis
- 🌐 **Propagation** — ITU-R P.452, P.528, P.676 propagation models
- 🎲 **Monte Carlo** — Aggregate interference simulation (ITU-R SM.2028)
- 📋 **Contribution Summary** — ITU-R contribution draft generator

## References
- ITU-R P.452, P.528, P.676
- ITU-R M.1642, SM.2028
- RTCA DO-235B, DO-253, DO-260B
- Radio Regulations No. 4.10, Resolution 233
